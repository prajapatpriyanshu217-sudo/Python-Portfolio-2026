# 🤖 AI Career Assistant

An AI-powered career assistant that helps users improve their resumes and prepare for job applications.

## 🚀 Planned Features

- Resume Analyzer
- ATS Score Checker
- Resume Improvement Suggestions
- Job Description Matching
- Cover Letter Generator
- LinkedIn Profile Optimization
- Skill Gap Analysis

## 🛠️ Technologies

- Python
- NiceGUI
- FastAPI
- SQLAlchemy
- PostgreSQL
- NLP
- AI API
